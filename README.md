# web-screenshots

Public mirror for screenshots posted by the `PR Screenshots`
GitHub Action on [`troutline-co/web`](https://github.com/troutline-co/web).
Each run lives under `PR-<number>/run-<run-id>/` and is
referenced by a comment on the corresponding PR.

Do not commit code here — this repo exists only as image hosting
so `raw.githubusercontent.com` URLs render in PR comments
(which can't carry auth tokens for the private source repo).
